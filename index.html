Onyx can generate music in multiple ways:

Text to Music (Lyrics):
• Generates songs with vocals and lyrics
• Requires TWO inputs:
  - Prompt: Describe genre, mood, style (e.g., "Indie folk, melancholic, introspective")
  - Lyrics: Write the actual song lyrics with [verse], [chorus] tags

Text to Music (Instrumental):
• Creates instrumental tracks without vocals
• Just describe genre, tempo, and mood
• No lyrics input needed

Text to SFX:
• Generate sound effects from text description
• Example: "Thunder rolling in the distance"
• Great for video projects and game development
Convert audio recordings to text transcription:

1. Select "Speech to Text" from Generation Type
2. Upload an audio file containing speech
3. Tap Generate to transcribe
4. Copy the transcribed text from the result dialog

Accepted Audio Types: MP3, OGG, WAV, M4A, AAC

Tips:
• Clear audio with minimal background noise works best
• Supports multiple languages
• Results are saved in your Gallery under "Transcripts"
